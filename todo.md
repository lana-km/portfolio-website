# Moderne Webdesigner Portfolio Website - Entwicklungsplan

## Design Guidelines

### Design References (Primary Inspiration)
- **Awwwards.com**: Cutting-edge web design showcase
- **Behance.net**: Creative portfolio presentations
- **Style**: Modern Minimalism + Dark Mode + Bold Typography + Glassmorphism

### Color Palette
- Primary: #0F0F0F (Deep Black - background)
- Secondary: #1A1A1A (Charcoal - cards)
- Accent: #FF6B6B (Coral Red - CTAs and highlights)
- Accent 2: #4ECDC4 (Turquoise - secondary accents)
- Accent 3: #FFE66D (Yellow - creative highlights)
- Text: #FFFFFF (White), #A0A0A0 (Light Gray - secondary)

### Typography
- Heading1: Inter font-weight 800 (56px) - Hero titles
- Heading2: Inter font-weight 700 (42px) - Section headers
- Heading3: Inter font-weight 600 (28px) - Subsections
- Body/Normal: Inter font-weight 400 (16px)
- Body/Emphasis: Inter font-weight 600 (16px)
- Navigation: Inter font-weight 500 (14px)

### Key Component Styles
- **Buttons**: Gradient background (coral to turquoise), white text, 8px rounded, hover: scale 1.05
- **Cards**: Dark with glassmorphism effect, backdrop-blur, subtle border, 16px rounded
- **Forms**: Dark inputs with gradient bottom border, focus: coral accent glow
- **Animations**: Smooth 300ms transitions, parallax scrolling, fade-in on scroll

### Layout & Spacing
- Hero section: Full viewport height with animated gradient background
- Portfolio grid: 3 columns desktop, 2 tablet, 1 mobile, 32px gaps
- Section padding: 100px vertical, 80px mobile
- Card hover: Lift 12px with glow shadow, scale 1.02, 400ms transition
